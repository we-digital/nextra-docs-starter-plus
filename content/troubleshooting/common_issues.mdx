# Troubleshooting Overview

Welcome to the troubleshooting guide for nextra-docs-starter-plus. This section provides solutions for the most common issues you may encounter.

## Quick Troubleshooting Checklist

Before diving into specific issues, try these common fixes:

1. **Restart your development server** - Many issues are resolved by simply restarting
2. **Clear cache** - Remove `.next` folder and rebuild
3. **Check environment variables** - Ensure all required variables are set
4. **Verify dependencies** - Run `npm install` or `pnpm install`
5. **Check browser console** - Look for JavaScript errors in DevTools

## Most Common Issues

### Server Won't Start
```bash
# Try these commands in order:
pnpm install
pnpm dev
```

### Environment Variables Not Working
```bash
# Check your .env.local file exists and contains:
SESSION_SECRET=your-secret-key
AUTH_USERNAME=your-username
AUTH_PASSWORD=your-password
```

### Search Not Working
```bash
# Rebuild the search index:
pnpm build
```

### Styles Not Loading
- Check if CSS files are properly imported in `app/layout.jsx`
- Clear browser cache (Ctrl+F5 or Cmd+Shift+R)
- Restart development server
