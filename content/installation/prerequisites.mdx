# Prerequisites

Before getting started with nextra-docs-starter-plus, ensure you have the following requirements met:

## System Requirements

### Node.js 18+
The template requires Node.js version 18 or higher. You can download it from [nodejs.org](https://nodejs.org/).

**Check your Node.js version:**
```bash
node --version
```

### Package Manager
Choose one of the following package managers:
- **npm** (comes with Node.js)
- **yarn** - Fast, reliable package manager
- **pnpm** - Efficient disk space usage (recommended)

### Git
Git is required for version control and cloning the repository.

**Check if Git is installed:**
```bash
git --version
```

If not installed, download from [git-scm.com](https://git-scm.com/).

## Knowledge Requirements

### Basic React
You should be familiar with:
- React components and JSX
- Hooks (useState, useEffect)
- Props and state management

### Next.js Fundamentals
Understanding of:
- File-based routing
- Pages and layouts
- API routes (optional)

### Markdown/MDX
Basic knowledge of:
- Markdown syntax
- MDX (Markdown + JSX)
- Frontmatter (optional)

## Development Environment

### Code Editor
Recommended editors with good TypeScript support:
- **Visual Studio Code** (recommended)
- **WebStorm**
- **Neovim** with TypeScript plugins

### Browser
Modern browser for development:
- **Chrome** (recommended for DevTools)
- **Firefox**
- **Safari**
- **Edge**

## Optional Tools

### Terminal/Command Line
Comfortable with basic command line operations:
- Navigating directories (`cd`, `ls`/`dir`)
- Running commands
- Environment variables

### Docker (Optional)
For containerized deployment:
- Docker Desktop
- Basic Docker knowledge

## Environment Setup

### Windows Users
Consider using:
- **Windows Subsystem for Linux (WSL)**
- **Git Bash**
- **PowerShell**

### macOS/Linux Users
Built-in terminal should work perfectly.

## Verification Checklist

Before proceeding, verify you have:

- [ ] Node.js 18+ installed
- [ ] Package manager available (npm/yarn/pnpm)
- [ ] Git installed and configured
- [ ] Code editor ready
- [ ] Basic React/Next.js knowledge
- [ ] Terminal/command line access

## Need Help?

If you're missing any prerequisites:

- **Node.js**: [Installation Guide](https://nodejs.org/en/download/)
- **Git**: [Getting Started Guide](https://git-scm.com/book/en/v2/Getting-Started-Installing-Git)
- **React**: [React Tutorial](https://react.dev/learn)
- **Next.js**: [Next.js Tutorial](https://nextjs.org/learn)

Once you have all prerequisites ready, proceed to [Quick Start](/quick-start) to begin setting up your documentation site!
